# PWA Icon Generation

To complete the PWA setup, you need to generate icon files:

1. **icon-192.png** - 192x192 pixels
2. **icon-512.png** - 512x512 pixels

You can:
- Use an online tool like https://realfavicongenerator.net/
- Use the provided icon.svg as a base
- Create icons with a clock/timesheet theme

Place the generated PNG files in the `/public` folder.
